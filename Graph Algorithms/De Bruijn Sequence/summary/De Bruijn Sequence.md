# De Bruijn Sequence

**CSES ID:** 1692  
**Full statement:** https://cses.fi/problemset/task/1692  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Build the shortest binary string that contains every length-$n$ binary substring as a contiguous substring.

## Input
A single integer $n$.

## Output
A minimum-length bit string covering all length-$n$ substrings.

## Constraints
- $1 \le n \le 15$
