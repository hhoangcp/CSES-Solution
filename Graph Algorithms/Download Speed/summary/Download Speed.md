# Download Speed

**CSES ID:** 1694  
**Full statement:** https://cses.fi/problemset/task/1694  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

A directed network of $n$ computers with $m$ capacity-limited links. Compute the maximum download speed from computer $1$ to computer $n$.

## Input
$n$ and $m$, then $m$ lines each giving a directed edge $a \to b$ with capacity $c$.

## Output
One integer: the maximum download speed.

## Constraints
- $1 \le n \le 500$
- $1 \le m \le 1000$
- $1 \le a,b \le n$
- $1 \le c \le 10^9$
