# School Dance

**CSES ID:** 1696  
**Full statement:** https://cses.fi/problemset/task/1696  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

There are $n$ boys and $m$ girls with $k$ allowed boy-girl pairs. Find the maximum number of pairs and list them.

## Input
$n$, $m$, and $k$, then $k$ lines each giving a potential pair $a, b$.

## Output
The maximum number of pairs $r$, then $r$ matching pairs.

## Constraints
- $1 \le n,m \le 500$
- $1 \le k \le 1000$
- $1 \le a \le n$
- $1 \le b \le m$
