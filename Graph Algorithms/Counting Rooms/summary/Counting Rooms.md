# Counting Rooms

**CSES ID:** 1192  
**Full statement:** https://cses.fi/problemset/task/1192  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given an $n \times m$ grid of floor (`.`) and wall (`#`) cells, count the connected floor regions using 4-directional adjacency.

## Input
Two integers $n, m$, then $n$ lines of $m$ characters (`.` or `#`).

## Output
A single integer: the number of distinct floor regions.

## Constraints
- $1 \le n,m \le 1000$
