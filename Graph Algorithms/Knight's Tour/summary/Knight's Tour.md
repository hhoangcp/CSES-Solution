# Knight's Tour

**CSES ID:** 1689  
**Full statement:** https://cses.fi/problemset/task/1689  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given a knight's starting square on an $8 \times 8$ board, produce a sequence of standard knight moves that visits every square exactly once.

## Input
Two integers $x, y$: the knight's starting position.

## Output
An $8 \times 8$ grid numbering each square with its visit order.

## Constraints
- $1 \le x,y \le 8$
