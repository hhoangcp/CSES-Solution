# Two Knights

**CSES ID:** 1072  
**Full statement:** https://cses.fi/problemset/task/1072  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

For each board size $k=1,2,\ldots,n$, count the number of ways to place two knights on a $k\times k$ chessboard so that neither attacks the other.

## Input
A single integer $n$.

## Output
Print $n$ integers: the answer for each $k$ from 1 to $n$.

## Constraints
- $1 \le n \le 10000$
