# String Reorder

**CSES ID:** 1743  
**Full statement:** https://cses.fi/problemset/task/1743  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Reorder the characters of an uppercase string so that no two adjacent characters match, choosing the lexicographically smallest such rearrangement. If no such string exists, print $-1$.

## Input
A single line with a string of length $n$ over letters A–Z.

## Output
Print the lexicographically smallest rearrangement with no equal adjacent characters, or $-1$ if impossible.

## Constraints
- $1 \le n \le 10^6$
