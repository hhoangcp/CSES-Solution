# Knight Moves Grid

**CSES ID:** 3217  
**Full statement:** https://cses.fi/problemset/task/3217  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

On an $n\times n$ chessboard with a knight, compute for every square the fewest knight moves needed to reach the top-left corner, which has distance 0. Output the whole distance grid.

## Input
A single integer $n$.

## Output
Print $n$ lines, each containing $n$ space-separated integers giving the move count for every square.

## Constraints
- $4 \le n \le 1000$
