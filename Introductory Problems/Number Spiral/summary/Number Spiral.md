# Number Spiral

**CSES ID:** 1071  
**Full statement:** https://cses.fi/problemset/task/1071  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

An infinite spiral grid starts with 1 in the upper-left corner and winds outward. Answer many queries asking for the number in row $y$, column $x$ of this spiral.

## Input
First line: integer $t$. Then $t$ lines each with two integers $y$ and $x$.

## Output
For each query print the value at row $y$, column $x$ of the spiral.

## Constraints
- $1 \le t \le 10^5$
- $1 \le y, x \le 10^9$
