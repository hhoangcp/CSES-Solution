# Mex Grid Construction

**CSES ID:** 3419  
**Full statement:** https://cses.fi/problemset/task/3419  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Build an $n\times n$ grid where each cell holds the smallest nonnegative integer that appears in neither the cells to its left in the same row nor the cells above it in the same column (the mex of those prefixes). Print the resulting grid.

## Input
A single integer $n$.

## Output
Print $n$ lines, each with $n$ space-separated integers forming the grid.

## Constraints
- $1 \le n \le 100$
