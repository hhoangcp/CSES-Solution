# Increasing Array

**CSES ID:** 1094  
**Full statement:** https://cses.fi/problemset/task/1094  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given an array of $n$ integers, make it non-decreasing using the fewest operations, where one operation increases a single element by 1.

## Input
First line: integer $n$. Second line: $n$ integers $x_1,x_2,\ldots,x_n$.

## Output
Print the minimum number of increment operations needed.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $1 \le x_i \le 10^9$
