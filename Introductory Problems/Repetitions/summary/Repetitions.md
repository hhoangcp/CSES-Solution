# Repetitions

**CSES ID:** 1069  
**Full statement:** https://cses.fi/problemset/task/1069  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given a DNA string over A, C, G, T, find the length of its longest run: a maximal substring consisting of a single repeated character.

## Input
A single string of $n$ characters from {A, C, G, T}.

## Output
Print one integer: the length of the longest single-character run.

## Constraints
- $1 \le n \le 10^6$
