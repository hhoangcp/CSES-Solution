# Apple Division

**CSES ID:** 1623  
**Full statement:** https://cses.fi/problemset/task/1623  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given $n$ apples each with a weight, split them into two groups so the difference between the two group totals is as small as possible. The answer is that minimum achievable difference.

## Input
First line: integer $n$. Second line: $n$ integers $p_1,p_2,\dots,p_n$, the apple weights.

## Output
Print the minimum possible difference between the two groups' total weights.

## Constraints
- $1 \le n \le 20$
- $1 \le p_i \le 10^9$
