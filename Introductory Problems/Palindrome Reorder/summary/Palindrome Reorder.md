# Palindrome Reorder

**CSES ID:** 1755  
**Full statement:** https://cses.fi/problemset/task/1755  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Rearrange the letters of a given uppercase string to form a palindrome. Output any valid palindrome, or `NO SOLUTION` when impossible.

## Input
A single string of length $n$ with letters A–Z.

## Output
Print a palindromic rearrangement, or `NO SOLUTION` if none exists.

## Constraints
- $1 \le n \le 10^6$
