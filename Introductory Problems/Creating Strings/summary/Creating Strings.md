# Creating Strings

**CSES ID:** 1622  
**Full statement:** https://cses.fi/problemset/task/1622  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given a string, produce every distinct permutation of its characters. Output the count of such permutations followed by the strings themselves in lexicographic order.

## Input
A single string of length $n$ with lowercase letters a–z.

## Output
First line: integer $k$, the number of distinct permutations. Then $k$ lines listing them in alphabetical order.

## Constraints
- $1 \le n \le 8$
