# Two Sets

**CSES ID:** 1092  
**Full statement:** https://cses.fi/problemset/task/1092  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Partition the integers $1,2,\ldots,n$ into two sets with equal total sum. If feasible print `YES` and an example partition; otherwise print `NO`.

## Input
A single integer $n$.

## Output
Print `YES` or `NO`. If `YES`, print the size and elements of the first set on two lines, then the same for the second set.

## Constraints
- $1 \le n \le 10^6$
