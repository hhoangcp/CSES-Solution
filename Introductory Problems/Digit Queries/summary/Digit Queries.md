# Digit Queries

**CSES ID:** 2431  
**Full statement:** https://cses.fi/problemset/task/2431  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Concatenate the positive integers in order into one infinite string "123456789101112...". Answer $q$ queries, each asking for the single digit at 1-indexed position $k$ in this string.

## Input
First line: integer $q$. Then $q$ lines each with an integer $k$.

## Output
For each query print the digit at position $k$.

## Constraints
- $1 \le q \le 1000$
- $1 \le k \le 10^{18}$
