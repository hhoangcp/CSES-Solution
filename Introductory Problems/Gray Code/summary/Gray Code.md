# Gray Code

**CSES ID:** 2205  
**Full statement:** https://cses.fi/problemset/task/2205  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

A Gray code of length $n$ lists all $2^n$ bit strings of length $n$ so that consecutive strings differ in exactly one bit. Construct and print any valid Gray code for the given $n$.

## Input
A single integer $n$.

## Output
Print $2^n$ lines, one bit string of length $n$ per line, forming a valid Gray code.

## Constraints
- $1 \le n \le 16$
