# Gray Code

**CSES ID:** 2205  
**Đề bài gốc:** https://cses.fi/problemset/task/2205  
**Giới hạn thời gian:** 1.00s | **Giới hạn bộ nhớ:** 512MB

> Tóm tắt đề bài.

Mã Gray độ dài $n$ liệt kê tất cả $2^n$ chuỗi bit độ dài $n$ sao cho hai chuỗi liên tiếp nhau khác nhau đúng một bit. Hãy xây dựng và in ra một mã Gray hợp lệ bất kỳ cho $n$ cho trước.

## Đầu vào
Một dòng chứa số nguyên $n$.

## Kết quả
In ra $2^n$ dòng, mỗi dòng một chuỗi bit độ dài $n$, tạo thành mã Gray hợp lệ.

## Ràng buộc
- $1 \le n \le 16$
