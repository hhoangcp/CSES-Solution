# Trailing Zeros

**CSES ID:** 1618  
**Full statement:** https://cses.fi/problemset/task/1618  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Compute how many zeros end the decimal representation of $n!$.

## Input
A single integer $n$.

## Output
Print the number of trailing zeros of $n!$.

## Constraints
- $1 \le n \le 10^9$
