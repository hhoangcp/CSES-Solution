# Bit Strings

**CSES ID:** 1617  
**Full statement:** https://cses.fi/problemset/task/1617  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Count how many binary strings of length $n$ exist. The answer must be output modulo $10^9+7$.

## Input
A single integer $n$.

## Output
Print $2^n$ modulo $10^9+7$.

## Constraints
- $1 \le n \le 10^6$
