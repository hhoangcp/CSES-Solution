# Permutations

**CSES ID:** 1070  
**Full statement:** https://cses.fi/problemset/task/1070  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

A permutation of $1,2,\ldots,n$ is "beautiful" when no two adjacent values differ by exactly 1. For given $n$, output any beautiful permutation, or `NO SOLUTION` if none exists.

## Input
A single integer $n$.

## Output
Print a beautiful permutation of $1,2,\ldots,n$, or `NO SOLUTION` if none exists.

## Constraints
- $1 \le n \le 10^6$
