# Missing Number

**CSES ID:** 1083  
**Full statement:** https://cses.fi/problemset/task/1083  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

You are given $n-1$ distinct integers from the range $1$ to $n$, so exactly one value is absent. Find that missing value.

## Input
First line: integer $n$. Second line: $n-1$ distinct integers in $[1,n]$.

## Output
Print the single missing integer.

## Constraints
- $2 \le n \le 2 \cdot 10^5$
