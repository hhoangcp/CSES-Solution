# Chessboard and Queens

**CSES ID:** 1624  
**Full statement:** https://cses.fi/problemset/task/1624  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Place eight queens on an $8\times 8$ board so that no two attack each other. Some squares are reserved (`*`) and may not hold a queen, though reserved squares do not block attacks. Count the number of valid placements.

## Input
Eight lines, each with eight characters: `.` for a free square and `*` for a reserved one.

## Output
Print one integer: the number of valid placements of the eight queens.

## Constraints
- Board size fixed at $8 \times 8$
