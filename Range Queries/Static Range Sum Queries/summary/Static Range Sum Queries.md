# Static Range Sum Queries

**CSES ID:** 1646  
**Full statement:** https://cses.fi/problemset/task/1646  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given an array, answer many queries each asking for the sum of values in a range $[a,b]$.

## Input
First line gives $n$ and $q$. Second line lists the array values. The following $q$ lines each give two endpoints $a$ and $b$.

## Output
Print the sum of the range for each query.

## Constraints
- $1 \le n,q \le 2 \cdot 10^5$
- $1 \le x_i \le 10^9$
- $1 \le a \le b \le n$
