# Maximum Xor Subarray

**CSES ID:** 1655  
**Full statement:** https://cses.fi/problemset/task/1655  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given an array, find the largest xor value achievable as the xor of some contiguous subarray.

## Input
First line: integer $n$. Second line: $n$ integers $x_1,\dots,x_n$.

## Output
One integer: the maximum subarray xor.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $0 \le x_i \le 10^9$
