# Number of Subset Xors

**CSES ID:** 3211  
**Full statement:** https://cses.fi/problemset/task/3211  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given an array, determine how many distinct values can be produced as the xor of some subset (the empty subset yields $0$).

## Input
First line: integer $n$. Second line: $n$ integers $x_1,\dots,x_n$.

## Output
One integer: the number of distinct subset xor values.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $0 \le x_i \le 10^9$
