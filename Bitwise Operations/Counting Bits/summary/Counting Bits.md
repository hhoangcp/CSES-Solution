# Counting Bits

**CSES ID:** 1146  
**Full statement:** https://cses.fi/problemset/task/1146  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given a positive integer $n$, determine the total number of set bits (ones) across the binary representations of all integers from $1$ through $n$.

## Input
A single integer $n$.

## Output
The total count of one bits in the binary forms of $1, 2, \dots, n$.

## Constraints
- $1 \le n \le 10^{15}$
