**Lời giải đầy đủ CSES Problem Set — 400 bài**

400 lời giải đã Accepted bằng C++ và Python cho [CSES Problem Set](https://cses.fi/problemset/), mỗi bài kèm lời giải chi tiết tiếng Anh và tiếng Việt.

> **Lưu ý:** Kho lưu trữ này chỉ dành cho mục đích tham khảo và học tập. Vui lòng không sao chép lời giải để gian lận trong các khóa học, cuộc thi hoặc bài kiểm tra. Tôi không chịu trách nhiệm về bất kỳ hành vi gian lận học tập nào phát sinh từ việc sử dụng kho lưu trữ này.

> **Phát hiện lỗi?** Nếu bạn thấy lỗi trong code hoặc lời giải, vui lòng mở [issue](https://github.com/hhoangcp/CSES-Solution/issues) hoặc liên hệ với tôi.

> **Ghi chú:** Phần lời giải được AI tạo dựa trên code đã Accepted (AC) của tôi, có thể còn lỗi — vui lòng báo lại nếu phát hiện.
