# Tree Traversals

**CSES ID:** 1702  
**Full statement:** https://cses.fi/problemset/task/1702  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given the preorder and inorder traversals of a binary tree whose $n$ nodes have distinct labels, recover the tree and print its postorder traversal.

## Input
First line: $n$. Then two lines: the preorder traversal and the inorder traversal, each with $n$ integers. The input corresponds to a valid binary tree.

## Output
Print the postorder traversal of the tree on one line.

## Constraints
- $1 \le n \le 10^5$
