# Distinct Sums Grid

**CSES ID:** 3424  
**Full statement:** https://cses.fi/problemset/task/3424  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Build an $n \times n$ grid filled with values from $1$ to $n$, where each value appears exactly $n$ times. Additionally, the $2n$ sums obtained from all rows and all columns must be pairwise distinct.

## Input
A single integer $n$.

## Output
Print any $n \times n$ grid satisfying the conditions, or `IMPOSSIBLE` if none exists.

## Constraints
- $1 \le n \le 1000$
