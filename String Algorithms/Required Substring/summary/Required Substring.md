# Required Substring

**CSES ID:** 1112  
**Full statement:** https://cses.fi/problemset/task/1112  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Count the number of length-$n$ strings over the uppercase alphabet A–Z that contain a given pattern as a contiguous substring. The answer is taken modulo $10^9+7$.

## Input
The first line contains the integer $n$. The second line contains the pattern of length $m$.

## Output
Print one integer: the number of valid strings modulo $10^9+7$.

## Constraints
- $1 \le n \le 1000$
- $1 \le m \le 100$
