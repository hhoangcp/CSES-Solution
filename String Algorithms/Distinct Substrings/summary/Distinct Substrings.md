# Distinct Substrings

**CSES ID:** 2105  
**Full statement:** https://cses.fi/problemset/task/2105  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

For a given string, determine how many different substrings it contains.

## Input
A single line with a string of length $n$ over the alphabet a–z.

## Output
Print one integer: the count of distinct substrings.

## Constraints
- $1 \le n \le 10^5$
