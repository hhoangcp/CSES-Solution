# String Matching

**CSES ID:** 1753  
**Full statement:** https://cses.fi/problemset/task/1753  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given a text and a single pattern, both over a–z, count the number of starting positions at which the pattern occurs in the text.

## Input
The first line contains the text of length $n$, and the second line contains the pattern of length $m$. Both use characters a–z.

## Output
Print one integer: the number of occurrences of the pattern in the text.

## Constraints
- $1 \le n, m \le 10^6$
