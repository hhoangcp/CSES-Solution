# Substring Distribution

**CSES ID:** 2110  
**Full statement:** https://cses.fi/problemset/task/2110  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

For a given string, count how many distinct substrings exist of each length from $1$ to $n$.

## Input
A single line with a string of length $n$ over characters a–z.

## Output
Print $n$ integers: the number of distinct substrings of lengths $1, 2, \dots, n$, space-separated.

## Constraints
- $1 \le n \le 10^5$
