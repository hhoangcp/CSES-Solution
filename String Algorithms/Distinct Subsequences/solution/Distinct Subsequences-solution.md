# Distinct Subsequences - Solution

**ID:** 1149 | **URL:** https://cses.fi/problemset/task/1149

## Problem Statement

Given a string $s$ of length $n$, how many different strings can you generate by removing some characters (keeping the order of the remaining ones)? Print the result modulo $10^9+7$ (the empty string is not counted).

## Approach

### Step 1: DP definition

Let $dp[i]$ be the number of distinct subsequences of the prefix $s[1..i]$, **including the empty subsequence** (for a cleaner formula).

**Base case:** $dp[0] = 1$ (only the empty subsequence).

### Step 2: Recurrence and removing duplicates

When adding the character $c = s[i]$, each subsequence of $s[1..i-1]$ has two choices: keep it as-is, or append $c$. The naive formula is $dp[i] = 2 \cdot dp[i-1]$. However, appending $c$ at position $i$ recreates subsequences ending in $c$ that were already formed at a previous occurrence of $c$, causing double-counting.

Let $j$ be the most recent position where $c$ appeared before $i$ (no such $j$ if $c$ never appeared). Set $\text{pre}[c] = j - 1$, so $dp[\text{pre}[c]] = dp[j-1]$ is the number of distinct subsequences of the prefix $s[1..j-1]$.

**The overcount equals exactly $dp[\text{pre}[c]]$.** Indeed, each subsequence $w$ of $s[1..j-1]$ gives $w + c$ when $c$ is appended; this same string is also produced by taking $w$ as a subsequence of $s[1..i-1]$ and appending $c$ at position $i$ — two ways to form the same string, hence a duplicate. Conversely, every string $w + c$ that was already counted (without using position $i$) takes its character $c$ from some position $\le j$, so $w$ is a subsequence of $s[1..j-1]$. Thus the number of duplicates is exactly $dp[j-1] = dp[\text{pre}[c]]$.

The recurrence is:

$$dp[i] = 2 \cdot dp[i-1] - dp[\text{pre}[s[i]]]$$

(when $\text{pre}[s[i]] = -1$, i.e. $c$ never appeared before, there is no duplicate, so $dp[i] = 2 \cdot dp[i-1]$).

Because of the subtraction, take the modulo carefully to avoid negative values (add $10^9+7$ before taking the remainder).

### Step 3: Final answer

$dp[n]$ counts all distinct subsequences **including the empty one**. The problem asks for the strings generated by deleting characters (excluding the empty string), so the answer is:

$$\text{Answer} = (dp[n] - 1) \bmod (10^9 + 7)$$

### Example

Using the example from the problem statement: $s = \text{"aybabtu"}$ ($n = 7$).

| $i$ | $s[i]$ | $dp[i-1]$ | $2 \cdot dp[i-1]$ | $\text{pre}[s[i]]$ | $dp[\text{pre}]$ | $dp[i]$ |
|---|---|---|---|---|---|---|
| 0 | — | — | — | — | — | 1 |
| 1 | a | 1 | 2 | $-1$ | — | 2 |
| 2 | y | 2 | 4 | $-1$ | — | 4 |
| 3 | b | 4 | 8 | $-1$ | — | 8 |
| 4 | a | 8 | 16 | $0$ | $1$ | 15 |
| 5 | b | 15 | 30 | $2$ | $4$ | 26 |
| 6 | t | 26 | 52 | $-1$ | — | 52 |
| 7 | u | 52 | 104 | $-1$ | — | 104 |

Some notable steps:

- $i = 1, 2, 3$: the characters `a`, `y`, `b` appear for the first time, so $\text{pre} = -1$ and $dp[i] = 2 \cdot dp[i-1]$.
- $i = 4$ (`a` repeats): the previous `a` is at $j = 1$, $\text{pre} = 0$, $dp[4] = 16 - dp[0] = 16 - 1 = 15$. The duplicate is the empty subsequence $+\text{"a"}$ = $\text{"a"}$ (already present).
- $i = 5$ (`b` repeats): the previous `b` is at $j = 3$, $\text{pre} = 2$, $dp[5] = 30 - dp[2] = 30 - 4 = 26$. The duplicates are the 4 subsequences of $\text{"ay"}$ with `b` appended: $\text{"b"}$, $\text{"ab"}$, $\text{"yb"}$, $\text{"ayb"}$ (all already present).
- $i = 6, 7$: `t`, `u` appear for the first time, so $dp$ only doubles.

$dp[7] = 104$ (including the empty subsequence). Answer: $104 - 1 = 103$.

Result:
```
103
```

## Complexity

- **Time:** $O(n)$ $\textemdash$ a single pass over the string, each position processed in $O(1)$.
- **Memory:** $O(n)$ $\textemdash$ the $dp$ array for DP states; the $\text{pre}$ array is a fixed 26 entries for characters 'a'–'z'.
