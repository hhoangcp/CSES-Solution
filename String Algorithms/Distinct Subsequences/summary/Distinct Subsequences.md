# Distinct Subsequences

**CSES ID:** 1149  
**Full statement:** https://cses.fi/problemset/task/1149  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

You may delete any subset of characters from a given string while preserving the relative order of the survivors. Count how many distinct resulting strings can be formed, including the empty result. The answer is reported modulo a large prime.

## Input
A single line with a string of length $n$ using characters a–z.

## Output
Print one integer: the number of distinct obtainable strings modulo $10^9+7$.

## Constraints
- $1 \le n \le 5 \cdot 10^5$
