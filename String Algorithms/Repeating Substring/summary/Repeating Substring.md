# Repeating Substring

**CSES ID:** 2106  
**Full statement:** https://cses.fi/problemset/task/2106  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

A repeating substring is one that occurs at two or more distinct starting positions in the given string. Find a longest such substring; if several tie, any is acceptable, and if none repeats, output $-1$.

## Input
A single line with a string of length $n$ over characters a–z.

## Output
Print one longest repeating substring, or $-1$ if no substring repeats.

## Constraints
- $1 \le n \le 10^5$
