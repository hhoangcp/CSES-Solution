# All Palindromes

**CSES ID:** 3138  
**Full statement:** https://cses.fi/problemset/task/3138  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

You are given a string over the lowercase alphabet. For every ending position, you must report the length of the longest palindromic substring that terminates exactly at that position.

## Input
A single line containing a string of length $n$ with characters in a–z.

## Output
Print $n$ space-separated integers, one per position, giving the longest palindrome length ending there.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
