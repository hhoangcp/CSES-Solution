# Finding Periods

**CSES ID:** 1733  
**Full statement:** https://cses.fi/problemset/task/1733  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

A period length $p$ of a string means that the string is formed by repeating its length-$p$ prefix, allowing the final repetition to be partial. Given a string, list every such period length in increasing order.

## Input
A single line with a string of length $n$ over characters a–z.

## Output
Print all period lengths in increasing order, separated by spaces.

## Constraints
- $1 \le n \le 10^6$
