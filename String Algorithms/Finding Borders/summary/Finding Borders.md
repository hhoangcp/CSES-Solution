# Finding Borders

**CSES ID:** 1732  
**Full statement:** https://cses.fi/problemset/task/1732  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

A border of a string is a proper prefix that also equals a suffix of the same string. Given a string, list the lengths of all its borders in increasing order.

## Input
A single line with a string of length $n$ over characters a–z.

## Output
Print all border lengths in increasing order, separated by spaces.

## Constraints
- $1 \le n \le 10^6$
