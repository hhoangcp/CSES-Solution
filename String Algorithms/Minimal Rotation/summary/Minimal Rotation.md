# Minimal Rotation

**CSES ID:** 1110  
**Full statement:** https://cses.fi/problemset/task/1110  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

A rotation of a string is formed by moving some leading characters to the end (equivalently, taking a cyclic shift). Among all rotations of the given string, output the one that is lexicographically smallest.

## Input
A single line with a string of length $n$ over characters a–z.

## Output
Print the lexicographically smallest rotation of the input.

## Constraints
- $1 \le n \le 10^6$
