# Filled Subgrid Count II

**CSES ID:** 3414  
**Full statement:** https://cses.fi/problemset/task/3414  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given an $n \times n$ grid using the first $k$ uppercase letters, count for each letter how many *rectangular* subgrids consist entirely of that single letter.

## Input
First line gives $n$ and $k$. Then $n$ lines follow, each with $n$ uppercase letters describing the grid.

## Output
Print $k$ lines, one count per letter in order.

## Constraints
- $1 \le n \le 3000$
- $1 \le k \le 26$
