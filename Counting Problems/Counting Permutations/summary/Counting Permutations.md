# Counting Permutations

**CSES ID:** 1075  
**Full statement:** https://cses.fi/problemset/task/1075  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

A permutation of $1,2,\ldots,n$ is *beautiful* when no two neighboring entries differ by exactly $1$. Given $n$, count how many beautiful permutations exist, modulo $10^9+7$.

## Input
A single line containing the integer $n$.

## Output
Print the number of beautiful permutations modulo $10^9+7$.

## Constraints
- $1 \le n \le 1000$
