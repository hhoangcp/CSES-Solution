# All Letter Subgrid Count II

**CSES ID:** 3416  
**Full statement:** https://cses.fi/problemset/task/3416  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given an $n \times n$ grid using the first $k$ uppercase letters, count the number of *rectangular* subgrids that contain every one of the $k$ letters at least once.

## Input
First line gives $n$ and $k$. Then $n$ lines follow, each with $n$ uppercase letters describing the grid.

## Output
Print the total number of qualifying rectangular subgrids.

## Constraints
- $1 \le n \le 500$
- $1 \le k \le 26$
