# All Letter Subgrid Count I

**CSES ID:** 3415  
**Full statement:** https://cses.fi/problemset/task/3415  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

You are given an $n \times n$ grid filled with the first $k$ uppercase letters. Count how many *square* subgrids contain every one of those $k$ letters at least once. The answer can be large, so report it directly.

## Input
First line gives $n$ and $k$. Then $n$ lines follow, each with $n$ uppercase letters describing the grid.

## Output
Print the total number of qualifying square subgrids.

## Constraints
- $1 \le n \le 3000$
- $1 \le k \le 26$
