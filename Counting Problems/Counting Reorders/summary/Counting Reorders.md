# Counting Reorders

**CSES ID:** 2421  
**Full statement:** https://cses.fi/problemset/task/2421  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given a string of lowercase letters, count the number of distinct rearrangements in which no two adjacent characters are equal. The answer is modulo $10^9+7$.

## Input
A single line containing a string of length $n$ over the letters `a`–`z`.

## Output
Print one integer, the count of valid rearrangements modulo $10^9+7$.

## Constraints
- $1 \le n \le 5000$
