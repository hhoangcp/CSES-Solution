# Empty String

**CSES ID:** 1080  
**Đề bài gốc:** https://cses.fi/problemset/task/1080  
**Giới hạn thời gian:** 1.00s | **Giới hạn bộ nhớ:** 512MB

> Tóm tắt đề bài.

Cho xâu các chữ cái thường, bạn liên tục xóa một cặp ký tự liền kề giống nhau cho đến khi xâu rỗng. Hãy đếm số thứ tự xóa khác nhau để đạt được xâu rỗng, lấy modulo $10^9+7$.

## Đầu vào
Một dòng duy nhất chứa xâu độ dài $n$ gồm các chữ cái `a`–`z`.

## Kết quả
In một số nguyên, số cách làm rỗng xâu modulo $10^9+7$.

## Ràng buộc
- $1 \le n \le 500$
