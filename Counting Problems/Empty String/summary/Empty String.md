# Empty String

**CSES ID:** 1080  
**Full statement:** https://cses.fi/problemset/task/1080  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given a string of lowercase letters, you repeatedly delete any pair of equal adjacent characters until the string is empty. Count the number of distinct sequences of removals that achieve this, modulo $10^9+7$.

## Input
A single line containing a string of length $n$ over the letters `a`–`z`.

## Output
Print one integer, the number of ways to empty the string modulo $10^9+7$.

## Constraints
- $1 \le n \le 500$
