# Tournament Graph Distribution

**CSES ID:** 3232  
**Full statement:** https://cses.fi/problemset/task/3232  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

A tournament graph is a directed graph with exactly one directed edge between each pair of nodes. Given $n$, for each $k=1\dots n$ count how many tournament graphs on $n$ labeled nodes have exactly $k$ strongly connected components, modulo $10^9+7$.

## Input
A single line containing the integer $n$.

## Output
Print $n$ lines: for each $k=1\dots n$, the number of tournament graphs modulo $10^9+7$.

## Constraints
- $1 \le n \le 500$
