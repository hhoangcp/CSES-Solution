# Counting Sequences

**CSES ID:** 2228  
**Full statement:** https://cses.fi/problemset/task/2228  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Count the number of length-$n$ sequences whose entries are integers in $1 \dots k$ and in which every value from $1$ to $k$ appears at least once. The answer is modulo $10^9+7$.

## Input
A single line with two integers $n$ and $k$.

## Output
Print one integer, the count of such sequences modulo $10^9+7$.

## Constraints
- $1 \le k \le n \le 10^6$
