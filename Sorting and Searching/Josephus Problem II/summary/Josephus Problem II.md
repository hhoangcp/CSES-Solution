# Josephus Problem II

**CSES ID:** 2163  
**Full statement:** https://cses.fi/problemset/task/2163  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

There are $n$ children in a circle; repeatedly skip $k$ surviving children and remove the next one. Output the removal order.

## Input
One line with integers $n$ and $k$.

## Output
Print $n$ integers: the removal order.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $0 \le k \le 10^9$
