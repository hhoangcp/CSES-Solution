# Subarray Divisibility

**CSES ID:** 1662  
**Full statement:** https://cses.fi/problemset/task/1662  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Count the contiguous subarrays whose sum is divisible by $n$.

## Input
First line: integer $n$. Next line: $n$ integers $a_1,\ldots,a_n$.

## Output
Print one integer: the number of subarrays whose sum is divisible by $n$.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $-10^9 \le a_i \le 10^9$
