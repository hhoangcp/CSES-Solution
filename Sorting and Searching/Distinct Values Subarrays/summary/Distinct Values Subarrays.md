# Distinct Values Subarrays

**CSES ID:** 3420  
**Full statement:** https://cses.fi/problemset/task/3420  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Count the number of contiguous subarrays in which all elements are pairwise distinct.

## Input
First line: integer $n$. Second line: $n$ integers $x_1,\ldots,x_n$.

## Output
Print one integer: the number of subarrays with all distinct elements.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $1 \le x_i \le 10^9$
