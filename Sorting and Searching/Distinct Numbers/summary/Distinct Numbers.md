# Distinct Numbers

**CSES ID:** 1621  
**Full statement:** https://cses.fi/problemset/task/1621  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

You are given a list of $n$ integers and must report how many distinct values it contains.

## Input
First line: integer $n$. Second line: $n$ integers $x_1,\ldots,x_n$.

## Output
Print one integer: the number of distinct values.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $1 \le x_i \le 10^9$
