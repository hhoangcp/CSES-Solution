# Maximum Subarray Sum

**CSES ID:** 1643  
**Full statement:** https://cses.fi/problemset/task/1643  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given an array of $n$ integers (possibly negative), find the largest sum of any nonempty contiguous subarray.

## Input
First line: integer $n$. Second line: $n$ integers $x_1,\ldots,x_n$.

## Output
Print one integer: the maximum subarray sum.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $-10^9 \le x_i \le 10^9$
