# Josephus Problem I

**CSES ID:** 2162  
**Full statement:** https://cses.fi/problemset/task/2162  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

There are $n$ children numbered $1\ldots n$ in a circle. Repeatedly every other remaining child is removed until none are left. Output the order of removal.

## Input
One line with integer $n$.

## Output
Print $n$ integers: the removal order.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
