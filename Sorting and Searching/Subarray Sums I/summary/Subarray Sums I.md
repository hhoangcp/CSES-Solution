# Subarray Sums I

**CSES ID:** 1660  
**Full statement:** https://cses.fi/problemset/task/1660  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given an array of $n$ positive integers, count the contiguous subarrays whose sum equals $x$.

## Input
First line: integers $n$ and $x$. Next line: $n$ integers $a_1,\ldots,a_n$.

## Output
Print one integer: the number of subarrays with sum $x$.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $1 \le x, a_i \le 10^9$
