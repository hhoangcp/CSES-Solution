# Missing Coin Sum

**CSES ID:** 2183  
**Full statement:** https://cses.fi/problemset/task/2183  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

You have $n$ coins with positive integer values and must find the smallest positive sum that no subset can form.

## Input
First line: integer $n$. Second line: $n$ integers $x_1,\ldots,x_n$.

## Output
Print one integer: the smallest sum that cannot be formed.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $1 \le x_i \le 10^9$
