# Reading Books

**CSES ID:** 1631  
**Full statement:** https://cses.fi/problemset/task/1631  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Two people must each read all $n$ books from start to finish, and they cannot read the same book at the same time. With reading times $t_i$, find the minimum total time required.

## Input
First line: integer $n$. Second line: $n$ integers $t_1,\ldots,t_n$.

## Output
Print one integer: the minimum total time required.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $1 \le t_i \le 10^9$
