# Movie Festival II

**CSES ID:** 1632  
**Full statement:** https://cses.fi/problemset/task/1632  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

There are $n$ movies with known start and end times and $k$ club members who can each watch one movie at a time. Maximize the total number of movies watched across all members.

## Input
First line: integers $n$ and $k$. Then $n$ lines, each with start $a$ and end $b$.

## Output
Print one integer: the maximum total number of movies the members can watch.

## Constraints
- $1 \le k \le n \le 2 \cdot 10^5$
- $1 \le a < b \le 10^9$
