# Distinct Values Subarrays II

**CSES ID:** 2428  
**Full statement:** https://cses.fi/problemset/task/2428  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Count the number of contiguous subarrays that contain at most $k$ distinct values.

## Input
First line: integers $n$ and $k$. Next line: $n$ integers $x_1,\ldots,x_n$.

## Output
Print one integer: the number of such subarrays.

## Constraints
- $1 \le k \le n \le 2 \cdot 10^5$
- $1 \le x_i \le 10^9$
