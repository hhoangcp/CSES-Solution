# Distinct Values Subsequences

**CSES ID:** 3421  
**Full statement:** https://cses.fi/problemset/task/3421  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Count the number of subsequences (not necessarily contiguous) whose elements are pairwise distinct. The answer can be huge, so report it modulo $10^9+7$.

## Input
First line: integer $n$. Second line: $n$ integers $x_1,\ldots,x_n$.

## Output
Print one integer: the number of distinct-element subsequences modulo $10^9+7$.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $1 \le x_i \le 10^9$
