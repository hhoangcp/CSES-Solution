# Swap Game

**CSES ID:** 1670  
**Full statement:** https://cses.fi/problemset/task/1670  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

A $3 \times 3$ grid holds the numbers $1$ through $9$ in some order. A move swaps the values of two orthogonally adjacent cells. Find the minimum number of moves to reach the row-major ordered grid.

## Input
Three lines, each with three integers describing the starting grid.

## Output
Print the minimum number of moves to reach the sorted grid.

## Constraints
- The grid is a permutation of $1,\ldots,9$.
