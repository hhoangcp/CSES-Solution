# Bubble Sort Rounds I

**CSES ID:** 3151  
**Full statement:** https://cses.fi/problemset/task/3151  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Each round of bubble sort scans the array left to right, swapping any adjacent pair that is out of order. Given an array, determine how many such rounds are required until the array is sorted.

## Input
An integer $n$ followed by $n$ integers $x_1,\ldots,x_n$.

## Output
Print the number of bubble sort rounds needed.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $1 \le x_i \le 10^9$
