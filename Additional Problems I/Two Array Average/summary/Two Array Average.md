# Two Array Average

**CSES ID:** 3361  
**Full statement:** https://cses.fi/problemset/task/3361  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given two length-$n$ arrays, choose a nonempty prefix of each so that the average of all selected elements is maximized, and output the two prefix lengths.

## Input
An integer $n$, then array $a$ of length $n$, then array $b$ of length $n$.

## Output
Print the two chosen prefix sizes; accepted within $10^{-6}$ absolute/relative error.

## Constraints
- $1 \le n \le 10^5$
- $1 \le a_i, b_i \le 10^9$
