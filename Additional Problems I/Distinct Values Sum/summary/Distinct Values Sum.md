# Distinct Values Sum

**CSES ID:** 3150  
**Full statement:** https://cses.fi/problemset/task/3150  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

For an array $x_1,\ldots,x_n$, let $d(a,b)$ be the count of distinct values in the subarray from index $a$ to $b$. Compute the sum of $d(a,b)$ over all subarrays.

## Input
An integer $n$, then $n$ integers $x_1,\ldots,x_n$.

## Output
Print the required total sum.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $1 \le x_i \le 10^9$
