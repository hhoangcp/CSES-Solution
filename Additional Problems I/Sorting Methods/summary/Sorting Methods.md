# Sorting Methods

**CSES ID:** 1162  
**Full statement:** https://cses.fi/problemset/task/1162  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given a permutation of $1,\ldots,n$, compute the minimum number of operations to sort it using each of four methods: adjacent swaps, arbitrary swaps, moving any element to any position, and moving any element to the front. Print all four.

## Input
An integer $n$, then $n$ integers describing the permutation.

## Output
Print four integers: the minimal step counts for the four methods.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
