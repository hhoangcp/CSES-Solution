# Subsets with Fixed Average

**CSES ID:** 3302  
**Full statement:** https://cses.fi/problemset/task/3302  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Count non-empty subsets of an array whose average equals a given value $a$.

## Input
Integers $n$ and $a$, then $n$ integers $x_1,\ldots,x_n$.

## Output
Print the count modulo $10^9 + 7$.

## Constraints
- $1 \le n \le 500$
- $1 \le a \le 500$
- $1 \le x_i \le 500$
