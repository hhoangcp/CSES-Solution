# Advertisement

**CSES ID:** 1142  
**Full statement:** https://cses.fi/problemset/task/1142  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

A fence is built from $n$ unit-width vertical boards of given heights. You must place an axis-aligned rectangle against the fence using consecutive boards, where the rectangle's height cannot exceed any board it covers. Determine the largest possible area of such a rectangle.

## Input
An integer $n$ followed by $n$ integers giving the heights of the boards.

## Output
Print the maximum achievable rectangular area.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $1 \le k_i \le 10^9$
