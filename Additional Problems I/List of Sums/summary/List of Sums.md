# List of Sums

**CSES ID:** 2414  
**Full statement:** https://cses.fi/problemset/task/2414  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

List $A$ has $n$ positive integers; list $B$ is the multiset of pairwise sums of elements of $A$. Given $B$, reconstruct any valid $A$.

## Input
An integer $n$, then $\frac{n(n-1)}{2}$ integers forming list $B$.

## Output
Print the $n$ elements of $A$ in any order; any valid solution is accepted.

## Constraints
- $3 \le n \le 100$
- $1 \le k \le 10^9$
