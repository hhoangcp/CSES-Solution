# Distinct Values Splits

**CSES ID:** 3190  
**Full statement:** https://cses.fi/problemset/task/3190  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given an array, count how many ways it can be partitioned into contiguous segments where each segment contains only distinct values.

## Input
An integer $n$, then $n$ integers $x_1,\ldots,x_n$.

## Output
Print the number of valid partitions modulo $10^9 + 7$.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $1 \le x_i \le 10^9$
