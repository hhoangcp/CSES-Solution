# Special Substrings

**CSES ID:** 2186  
**Full statement:** https://cses.fi/problemset/task/2186  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

A substring is *special* when every character occurring in it appears the same number of times. Count the special substrings of a given lowercase string.

## Input
A single string of length $n$ over `a`...`z`.

## Output
Print the number of special substrings.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
