# Writing Numbers

**CSES ID:** 1086  
**Full statement:** https://cses.fi/problemset/task/1086  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

You write the integers $1,2,3,\ldots$ in order, but each decimal digit key may be pressed at most $n$ times overall. Find the largest integer you can complete before exceeding this budget.

## Input
A single integer $n$.

## Output
Print the last number that can be written within the key-press budget.

## Constraints
- $1 \le n \le 10^{18}$
