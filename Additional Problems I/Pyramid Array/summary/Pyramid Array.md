# Pyramid Array

**CSES ID:** 1747  
**Full statement:** https://cses.fi/problemset/task/1747  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given an array of $n$ distinct integers, rearrange it using adjacent swaps into a sequence that increases then decreases (a "pyramid", allowing purely monotone shapes). Minimize the number of adjacent swaps.

## Input
An integer $n$, then $n$ distinct integers $x_1,\ldots,x_n$.

## Output
Print the minimum number of adjacent swaps required.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $1 \le x_i \le 10^9$
