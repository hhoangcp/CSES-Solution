# Square Subsets

**CSES ID:** 3193  
**Full statement:** https://cses.fi/problemset/task/3193  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Count subsets of an array whose product is a perfect square, including the empty subset (product $1$).

## Input
An integer $n$, then $n$ integers $x_1,\ldots,x_n$.

## Output
Print the count modulo $10^9 + 7$.

## Constraints
- $1 \le n \le 5000$
- $1 \le x_i \le 5000$
