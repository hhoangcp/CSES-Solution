# Multiplication Table

**CSES ID:** 2422  
**Full statement:** https://cses.fi/problemset/task/2422  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Consider the $n \times n$ multiplication table whose $(i,j)$ entry is $i \cdot j$. With $n$ odd, sort all $n^2$ entries increasingly and report the middle element.

## Input
A single integer $n$.

## Output
Print the middle element of the sorted multiplication table.

## Constraints
- $1 \le n < 10^6$
