# Shortest Subsequence

**CSES ID:** 1087  
**Full statement:** https://cses.fi/problemset/task/1087  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given a DNA string over `A`, `C`, `G`, `T`, produce the shortest string over the same alphabet that is *not* a subsequence of the input.

## Input
A single DNA string of length $n$.

## Output
Print any shortest string that is not a subsequence of the input.

## Constraints
- $1 \le n \le 10^6$
