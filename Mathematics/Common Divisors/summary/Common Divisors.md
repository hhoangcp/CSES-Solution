# Common Divisors

**CSES ID:** 1081  
**Full statement:** https://cses.fi/problemset/task/1081  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given $n$ positive integers, pick a pair of distinct elements whose greatest common divisor is as large as possible. Output that maximum GCD.

## Input
The first line is $n$, the second line lists the $n$ array values.

## Output
Print the largest possible GCD of any chosen pair.

## Constraints
- $2 \le n \le 2 \cdot 10^5$
- $1 \le x_i \le 10^6$
