# Bracket Sequences I

**CSES ID:** 2064  
**Full statement:** https://cses.fi/problemset/task/2064  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Count the number of valid balanced bracket sequences of length $n$ (made of `(` and `)`). Return the count modulo $10^9+7$.

## Input
A single integer $n$.

## Output
Print the number of valid bracket sequences of length $n$ modulo $10^9+7$.

## Constraints
- $1 \le n \le 10^6$
