# Creating Strings II

**CSES ID:** 1715  
**Full statement:** https://cses.fi/problemset/task/1715  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given a string, count how many distinct strings can be formed by rearranging its characters. Answer modulo $10^9+7$.

## Input
A single line containing the string of length $n$ (lowercase a–z).

## Output
Print the number of distinct permutations modulo $10^9+7$.

## Constraints
- $1 \le n \le 10^6$
