# Counting Necklaces

**CSES ID:** 2209  
**Full statement:** https://cses.fi/problemset/task/2209  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Count the number of distinct necklaces formed by $n$ beads each colored with one of $m$ colors, where necklaces equivalent under rotation are considered the same. Answer modulo $10^9+7$.

## Input
A single line with two integers $n$ and $m$ (beads and colors).

## Output
Print the number of distinct necklaces modulo $10^9+7$.

## Constraints
- $1 \le n,m \le 10^6$
