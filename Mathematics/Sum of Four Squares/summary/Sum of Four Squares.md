# Sum of Four Squares

**CSES ID:** 3355  
**Full statement:** https://cses.fi/problemset/task/3355  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

For each given non-negative integer $n$, output four non-negative integers $a,b,c,d$ with $n = a^2+b^2+c^2+d^2$.

## Input
The first line is $t$. Each of the next $t$ lines contains a non-negative integer $n$.

## Output
For each test print four non-negative integers $a,b,c,d$ satisfying the identity (any valid quadruple).

## Constraints
- $1 \le t \le 1000$
- $0 \le n \le 10^7$
- $\sum n \le 10^7$
