# Exponentiation

**CSES ID:** 1095  
**Full statement:** https://cses.fi/problemset/task/1095  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Compute $a^b \bmod (10^9+7)$ for many queries. Here $0^0$ is defined to be $1$.

## Input
The first line is $n$, followed by $n$ lines each containing $a$ and $b$.

## Output
Print $a^b \bmod (10^9+7)$ for each query on its own line.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $0 \le a,b \le 10^9$
