# Next Prime

**CSES ID:** 3396  
**Full statement:** https://cses.fi/problemset/task/3396  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given a positive integer $n$, find the smallest prime number strictly greater than $n$.

## Input
The first line is $t$, followed by $t$ lines each containing one positive integer $n$.

## Output
For each test print the next prime strictly greater than $n$.

## Constraints
- $1 \le t \le 20$
- $1 \le n \le 10^{12}$
