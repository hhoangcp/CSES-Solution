# Throwing Dice

**CSES ID:** 1096  
**Full statement:** https://cses.fi/problemset/task/1096  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Count the number of ordered sequences of die throws (each yielding 1–6) whose outcomes sum to exactly $n$. Answer modulo $10^9+7$.

## Input
A single integer $n$.

## Output
Print the number of ways modulo $10^9+7$.

## Constraints
- $1 \le n \le 10^{18}$
