# Counting Coprime Pairs

**CSES ID:** 2417  
**Full statement:** https://cses.fi/problemset/task/2417  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given $n$ positive integers, count how many unordered pairs of them have GCD equal to 1 (coprime pairs).

## Input
The first line is $n$, the second line lists the $n$ integers.

## Output
Print the total number of coprime pairs.

## Constraints
- $1 \le n \le 10^5$
- $1 \le x_i \le 10^6$
