# Candy Lottery

**CSES ID:** 1727  
**Full statement:** https://cses.fi/problemset/task/1727  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Each of $n$ children independently draws a uniform random integer in $[1,k]$. Compute the expected value of the maximum drawn value, printed to six decimals.

## Input
A single line with two integers $n$ and $k$.

## Output
Print the expected maximum, rounded to six decimal places (round half to even).

## Constraints
- $1 \le n \le 100$
- $1 \le k \le 100$
