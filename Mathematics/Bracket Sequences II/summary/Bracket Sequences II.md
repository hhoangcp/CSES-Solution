# Bracket Sequences II

**CSES ID:** 2187  
**Full statement:** https://cses.fi/problemset/task/2187  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given a total length $n$ and a fixed prefix string of brackets, count how many valid balanced bracket sequences of length $n$ start with that prefix. Return the answer modulo $10^9+7$.

## Input
The first line contains $n$. The second line contains the given prefix string of length $k$.

## Output
Print the number of valid sequences modulo $10^9+7$.

## Constraints
- $1 \le k \le n \le 10^6$
