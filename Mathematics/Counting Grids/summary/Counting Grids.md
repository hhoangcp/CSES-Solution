# Counting Grids

**CSES ID:** 2210  
**Full statement:** https://cses.fi/problemset/task/2210  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Count the number of distinct $n \times n$ binary grids, where two grids are equivalent if one can be rotated (multiples of 90 degrees) to match the other. Answer modulo $10^9+7$.

## Input
A single integer $n$, the grid size.

## Output
Print the number of distinct grids modulo $10^9+7$.

## Constraints
- $1 \le n \le 10^9$
