# Triangle Number Sums

**CSES ID:** 3406  
**Full statement:** https://cses.fi/problemset/task/3406  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

A triangle number is $T_k = 1+2+\dots+k = k(k+1)/2$. For each given $n$, find the minimum number of triangle numbers (repetitions allowed) whose sum is $n$.

## Input
The first line is $t$. Each of the next $t$ lines contains a positive integer $n$.

## Output
For each test print the smallest number of triangle numbers needed.

## Constraints
- $1 \le t \le 100$
- $1 \le n \le 10^{12}$
