# Grundy's Game

**CSES ID:** 2207  
**Full statement:** https://cses.fi/problemset/task/2207  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

An impartial game starts with one heap of $n$ coins. A move splits a heap into two nonempty heaps of unequal sizes; the player making the last move wins. Determine the winner under optimal play.

## Input
The first line is $t$, followed by $t$ lines each containing one integer $n$.

## Output
For each test print `first` or `second`.

## Constraints
- $1 \le t \le 10^5$
- $1 \le n \le 10^6$
