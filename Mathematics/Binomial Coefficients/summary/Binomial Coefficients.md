# Binomial Coefficients

**CSES ID:** 1079  
**Full statement:** https://cses.fi/problemset/task/1079  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given many queries, compute the binomial coefficient $\binom{a}{b}$ modulo $10^9+7$ for each pair $(a,b)$.

## Input
The first line gives $n$, the number of queries. Each of the next $n$ lines contains two integers $a$ and $b$.

## Output
Print each binomial coefficient modulo $10^9+7$ on its own line.

## Constraints
- $1 \le n \le 10^5$
- $0 \le b \le a \le 10^6$
