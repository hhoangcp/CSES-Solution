# Distributing Apples

**CSES ID:** 1716  
**Full statement:** https://cses.fi/problemset/task/1716  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Count the number of ways to distribute $m$ identical apples among $n$ distinct children (children may receive zero). Answer modulo $10^9+7$.

## Input
A single line with two integers $n$ and $m$.

## Output
Print the number of distributions modulo $10^9+7$.

## Constraints
- $1 \le n,m \le 10^6$
