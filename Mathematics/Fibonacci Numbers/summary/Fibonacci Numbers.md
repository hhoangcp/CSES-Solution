# Fibonacci Numbers

**CSES ID:** 1722  
**Full statement:** https://cses.fi/problemset/task/1722  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given $n$ (which can be very large), compute the $n$-th Fibonacci number $F_n$ modulo $10^9+7$, where $F_0=0$, $F_1=1$ and $F_n=F_{n-1}+F_{n-2}$.

## Input
A single integer $n$.

## Output
Print $F_n \bmod (10^9+7)$.

## Constraints
- $0 \le n \le 10^{18}$
