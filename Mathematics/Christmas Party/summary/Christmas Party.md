# Christmas Party

**CSES ID:** 1717  
**Full statement:** https://cses.fi/problemset/task/1717  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Each of $n$ children brings one gift and must receive a gift brought by someone else (a derangement of gifts). Count the number of valid distributions modulo $10^9+7$.

## Input
A single integer $n$, the number of children.

## Output
Print the number of valid derangements modulo $10^9+7$.

## Constraints
- $1 \le n \le 10^6$
