# Counting Divisors

**CSES ID:** 1713  
**Full statement:** https://cses.fi/problemset/task/1713  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

For each of $n$ given integers $x$, output how many positive divisors $x$ has.

## Input
The first line is $n$. Each of the next $n$ lines contains one integer $x$.

## Output
For each $x$ print its number of divisors on its own line.

## Constraints
- $1 \le n \le 10^5$
- $1 \le x \le 10^6$
