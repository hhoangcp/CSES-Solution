# Polygon Lattice Points

**CSES ID:** 2193  
**Full statement:** https://cses.fi/problemset/task/2193  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given a simple polygon with integer-coordinate vertices, count how many lattice points lie strictly inside it and how many lie on its boundary.

## Input
First an integer $n$, then $n$ lines each with two integers $x_i, y_i$ giving a vertex of the polygon.

## Output
Print two integers: the number of interior lattice points and the number of boundary lattice points.

## Constraints
- $3 \le n \le 10^5$
- $-10^9 \le x_i, y_i \le 10^9$
