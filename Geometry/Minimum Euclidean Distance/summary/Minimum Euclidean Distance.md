# Minimum Euclidean Distance

**CSES ID:** 2194  
**Full statement:** https://cses.fi/problemset/task/2194  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given a set of distinct points in the plane, find the smallest Euclidean distance between any two of them. To stay in integer arithmetic you report $d^2$, the squared minimum distance.

## Input
First an integer $n$, then $n$ lines each with two integers $x$ and $y$ giving a distinct point.

## Output
Print one integer: $d^2$ where $d$ is the minimum Euclidean distance.

## Constraints
- $2 \le n \le 2 \cdot 10^5$
- $-10^9 \le x, y \le 10^9$
