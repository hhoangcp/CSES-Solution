# Polygon Area

**CSES ID:** 2191  
**Full statement:** https://cses.fi/problemset/task/2191  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

You are given a simple polygon of $n$ vertices listed in order. Compute its area, reporting $2a$ so the answer stays an integer.

## Input
First an integer $n$, then $n$ lines each with two integers $x_i, y_i$ giving a vertex of the polygon.

## Output
Print one integer: $2a$ where the polygon area is $a$.

## Constraints
- $3 \le n \le 1000$
- $-10^9 \le x_i, y_i \le 10^9$
