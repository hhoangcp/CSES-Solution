# Knight Moves Queries

**CSES ID:** 3218  
**Full statement:** https://cses.fi/problemset/task/3218  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

A knight moves on an infinite $1$-indexed chessboard. For each query giving a starting square $(x,y)$, find the minimum knight moves needed to reach the top-left corner $(1,1)$.

## Input
First line: $n$ queries. Then $n$ lines each with $x$ and $y$.

## Output
For each query, print the minimum number of moves.

## Constraints
- $1 \le n \le 10^5$
- $1 \le x, y \le 10^9$
