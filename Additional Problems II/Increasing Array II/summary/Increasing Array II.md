# Increasing Array II

**CSES ID:** 2132  
**Full statement:** https://cses.fi/problemset/task/2132  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given an array, change it into a non-decreasing array by repeatedly increasing or decreasing any element by 1 per move. Find the minimum total moves.

## Input
First line: $n$. Second line: $n$ integers $x_1,\dots,x_n$.

## Output
One integer: the minimum number of moves.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $1 \le x_i \le 10^9$
