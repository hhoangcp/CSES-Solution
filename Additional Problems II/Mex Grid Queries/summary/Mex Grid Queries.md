# Mex Grid Queries

**CSES ID:** 1157  
**Full statement:** https://cses.fi/problemset/task/1157  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

A $1$-indexed 2D grid is filled so that each cell holds the smallest nonnegative integer absent from its row to the left and its column above. Given a cell $(y,x)$, compute its value.

## Input
A single line with two integers $y$ and $x$.

## Output
One integer: the value at cell $(y,x)$.

## Constraints
- $1 \le y, x \le 10^9$
