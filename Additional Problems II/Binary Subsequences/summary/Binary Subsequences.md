# Binary Subsequences

**CSES ID:** 2430  
**Full statement:** https://cses.fi/problemset/task/2430  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given an integer $n$, construct a binary string of minimum length whose number of distinct subsequences is exactly $n$. Any valid shortest string is accepted.

## Input
A single line with the integer $n$.

## Output
One binary string of minimum length having exactly $n$ distinct subsequences.

## Constraints
- $1 \le n \le 10^6$
