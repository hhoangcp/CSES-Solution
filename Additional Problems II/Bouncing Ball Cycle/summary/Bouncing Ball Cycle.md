# Bouncing Ball Cycle

**CSES ID:** 3216  
**Full statement:** https://cses.fi/problemset/task/3216  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

A ball starts at the top-left of an $n \times m$ grid moving diagonally away from that corner and reflects off borders. Determine the number of steps until it first returns to the starting cell, and how many distinct cells it visits.

## Input
First line: $t$ tests. Then $t$ lines each with $n$ and $m$.

## Output
For each test, print two integers: the cycle length in steps and the number of distinct visited cells.

## Constraints
- $1 \le t \le 1000$
- $2 \le n, m \le 10^9$
