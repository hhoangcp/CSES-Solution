# Bit Substrings

**CSES ID:** 2115  
**Full statement:** https://cses.fi/problemset/task/2115  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given a binary string of length $n$, for each $k$ from $0$ to $n$ count how many non-empty contiguous substrings contain exactly $k$ ones.

## Input
A single line with a binary string of length $n$.

## Output
Print $n+1$ integers: the count for $k=0,1,\dots,n$.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
