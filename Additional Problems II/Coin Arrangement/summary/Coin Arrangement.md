# Coin Arrangement

**CSES ID:** 2180  
**Full statement:** https://cses.fi/problemset/task/2180  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

A $2 \times n$ grid holds a total of $2n$ coins spread across its cells. Move coins one step left/right/up/down until each cell has exactly one coin. Find the minimum total number of moves.

## Input
First line: $n$. Then two lines, each with $n$ integers giving the coin counts per column.

## Output
One integer: the minimum number of moves.

## Constraints
- $1 \le n \le 10^5$
