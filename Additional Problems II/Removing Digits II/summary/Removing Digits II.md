# Removing Digits II

**CSES ID:** 2174  
**Full statement:** https://cses.fi/problemset/task/2174  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Starting from an integer $n$, repeatedly subtract any one-digit number that appears in the current value, until reaching 0. Find the minimum number of steps.

## Input
A single line with the integer $n$.

## Output
One integer: the minimum number of steps.

## Constraints
- $1 \le n \le 10^{18}$
