# Dice Combinations

**CSES ID:** 1633  
**Full statement:** https://cses.fi/problemset/task/1633  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Count the number of ordered sequences of dice rolls (each roll in $[1,6]$, at least one roll) whose outcomes sum to $n$.

## Input
A single integer $n$.

## Output
The number of ways modulo $10^9+7$.

## Constraints
- $1 \le n \le 10^6$
