# Counting Numbers

**CSES ID:** 2220  
**Full statement:** https://cses.fi/problemset/task/2220  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Count how many integers in the inclusive range $[a,b]$ have no two identical adjacent digits.

## Input
Two integers $a$ and $b$.

## Output
The count of such integers.

## Constraints
- $0 \le a \le b \le 10^{18}$
