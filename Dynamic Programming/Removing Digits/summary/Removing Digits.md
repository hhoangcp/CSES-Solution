# Removing Digits

**CSES ID:** 1637  
**Full statement:** https://cses.fi/problemset/task/1637  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Starting from a positive integer $n$, each step subtracts one of the current number's digits. Find the minimum number of steps needed to reach $0$.

## Input
A single integer $n$.

## Output
The minimum number of steps.

## Constraints
- $1 \le n \le 10^6$
