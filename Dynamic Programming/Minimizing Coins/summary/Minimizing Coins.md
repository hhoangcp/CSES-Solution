# Minimizing Coins

**CSES ID:** 1634  
**Full statement:** https://cses.fi/problemset/task/1634  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given $n$ coin denominations, find the minimum number of coins needed to form exactly the sum $x$ (unlimited use of each denomination); output $-1$ if impossible.

## Input
Integers $n$ and $x$, then $n$ distinct coin values $c_i$.

## Output
The minimum coin count, or $-1$.

## Constraints
- $1 \le n \le 100$
- $1 \le x \le 10^6$
- $1 \le c_i \le 10^6$
