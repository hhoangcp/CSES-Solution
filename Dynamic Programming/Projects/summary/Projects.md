# Projects

**CSES ID:** 1140  
**Full statement:** https://cses.fi/problemset/task/1140  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

There are $n$ projects, each with a start day, end day, and reward. A person can do only one project per day (no overlapping intervals). Find the maximum total reward attainable from a non-overlapping selection.

## Input
An integer $n$, then $n$ lines of $a_i, b_i, p_i$ (start, end, reward).

## Output
The maximum reward.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $1 \le a_i \le b_i \le 10^9$
- $1 \le p_i \le 10^9$
