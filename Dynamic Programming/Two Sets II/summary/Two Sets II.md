# Two Sets II

**CSES ID:** 1093  
**Full statement:** https://cses.fi/problemset/task/1093  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Count the number of ways to partition the integers $1,2,\ldots,n$ into two sets with equal sum. The order of the two sets does not matter (count each partition once).

## Input
A single integer $n$.

## Output
The number of such partitions modulo $10^9+7$.

## Constraints
- $1 \le n \le 500$
