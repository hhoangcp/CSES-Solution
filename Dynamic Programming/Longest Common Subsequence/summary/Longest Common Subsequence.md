# Longest Common Subsequence

**CSES ID:** 3403  
**Full statement:** https://cses.fi/problemset/task/3403  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given two integer arrays, find their longest common subsequence (a sequence appearing in both, preserving order, with gaps allowed), and print both its length and one example.

## Input
Integers $n$ and $m$, then array $a$ of length $n$, then array $b$ of length $m$.

## Output
The length of the LCS on the first line, then the chosen subsequence on the next line.

## Constraints
- $1 \le n,m \le 1000$
- $1 \le a_i, b_i \le 10^9$
