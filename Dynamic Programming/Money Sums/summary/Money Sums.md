# Money Sums

**CSES ID:** 1745  
**Full statement:** https://cses.fi/problemset/task/1745  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given $n$ coins of known values, determine every distinct total sum that can be formed by choosing any subset of the coins.

## Input
An integer $n$, then $n$ coin values $x_1,\dots,x_n$.

## Output
First the count $k$ of distinct sums, then all sums in increasing order.

## Constraints
- $1 \le n \le 100$
- $1 \le x_i \le 1000$
