# Removal Game

**CSES ID:** 1097  
**Full statement:** https://cses.fi/problemset/task/1097  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Two players take turns taking either the first or last element of a list of $n$ numbers, adding it to their own score, both playing optimally. Compute the first player's maximum possible score.

## Input
An integer $n$, then $n$ integers $x_1,\dots,x_n$.

## Output
The first player's maximum score under optimal play.

## Constraints
- $1 \le n \le 5000$
- $-10^9 \le x_i \le 10^9$
