# Rectangle Cutting

**CSES ID:** 1744  
**Full statement:** https://cses.fi/problemset/task/1744  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Starting from an $a \times b$ rectangle, repeatedly split a chosen rectangle into two integer-sided rectangles. Find the minimum number of cuts needed to reduce the whole shape entirely into squares.

## Input
Two integers $a$ and $b$.

## Output
The minimum number of cuts.

## Constraints
- $1 \le a,b \le 500$
