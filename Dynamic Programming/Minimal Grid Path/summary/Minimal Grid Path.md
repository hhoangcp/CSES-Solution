# Minimal Grid Path

**CSES ID:** 3359  
**Full statement:** https://cses.fi/problemset/task/3359  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

On an $n \times n$ grid of uppercase letters, moving only right or down from the top-left to the bottom-right corner, find the lexicographically smallest string that can be formed along the path.

## Input
An integer $n$, then $n$ rows of $n$ letters (A–Z).

## Output
The lexicographically minimal string.

## Constraints
- $1 \le n \le 3000$
