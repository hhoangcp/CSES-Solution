# Minimal Grid Path

**CSES ID:** 3359  
**Đề bài gốc:** https://cses.fi/problemset/task/3359  
**Giới hạn thời gian:** 1.00s | **Giới hạn bộ nhớ:** 512MB

> Tóm tắt đề bài.

Trên lưới $n \times n$ các chữ cái hoa, chỉ đi sang phải hoặc xuống dưới từ góc trên trái đến góc dưới phải, hãy tìm chuỗi có thứ tự từ điển nhỏ nhất tạo được trên đường đi.

## Đầu vào
Một số nguyên $n$, rồi $n$ dòng mỗi dòng $n$ chữ cái (A–Z).

## Kết quả
Chuỗi có thứ tự từ điển nhỏ nhất.

## Ràng buộc
- $1 \le n \le 3000$
