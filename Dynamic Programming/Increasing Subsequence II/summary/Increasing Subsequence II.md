# Increasing Subsequence II

**CSES ID:** 1748  
**Full statement:** https://cses.fi/problemset/task/1748  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given an array of $n$ integers, count the total number of increasing subsequences, where subsequences with identical values but at different positions are counted separately.

## Input
An integer $n$, then $n$ integers $x_1,\dots,x_n$.

## Output
The number of increasing subsequences modulo $10^9+7$.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $1 \le x_i \le 10^9$
