# Counting Tilings

**CSES ID:** 2181  
**Full statement:** https://cses.fi/problemset/task/2181  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Count the number of ways to tile an $n \times m$ rectangle using $1 \times 2$ and $2 \times 1$ dominoes.

## Input
Two integers $n$ and $m$.

## Output
The number of tilings modulo $10^9+7$.

## Constraints
- $1 \le n \le 10$
- $1 \le m \le 1000$
