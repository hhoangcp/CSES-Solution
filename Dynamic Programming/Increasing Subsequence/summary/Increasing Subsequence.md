# Increasing Subsequence

**CSES ID:** 1145  
**Full statement:** https://cses.fi/problemset/task/1145  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given an array of $n$ integers, find the length of its longest strictly increasing subsequence (obtained by deleting elements without reordering).

## Input
An integer $n$, then $n$ integers $x_1,\dots,x_n$.

## Output
The length of the longest increasing subsequence.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $1 \le x_i \le 10^9$
