# Edit Distance

**CSES ID:** 1639  
**Full statement:** https://cses.fi/problemset/task/1639  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Compute the minimum number of single-character insertions, deletions, or replacements needed to transform one uppercase string into another.

## Input
Two strings of lengths $n$ and $m$ over A–Z, one per line.

## Output
The edit distance between the two strings.

## Constraints
- $1 \le n,m \le 5000$
