# Edit Distance

**CSES ID:** 1639  
**Đề bài gốc:** https://cses.fi/problemset/task/1639  
**Giới hạn thời gian:** 1.00s | **Giới hạn bộ nhớ:** 512MB

> Tóm tắt đề bài.

Tính số phép toán ít nhất (thêm, xóa, thay thế một ký tự) để biến đổi một chuỗi chữ hoa thành chuỗi kia.

## Đầu vào
Hai chuỗi độ dài $n$ và $m$ trên A–Z, mỗi chuỗi một dòng.

## Kết quả
Khoảng cách chỉnh sửa giữa hai chuỗi.

## Ràng buộc
- $1 \le n,m \le 5000$
