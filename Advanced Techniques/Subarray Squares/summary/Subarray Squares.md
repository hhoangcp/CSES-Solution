# Subarray Squares

**CSES ID:** 2086  
**Full statement:** https://cses.fi/problemset/task/2086  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Split an array of $n$ elements into exactly $k$ contiguous subarrays. The cost of a subarray is the square of the sum of its elements. Find the minimum possible total cost.

## Input
First line: $n$ and $k$. Second line: $n$ integers $x_1,\ldots,x_n$.

## Output
Print the minimum total cost.

## Constraints
- $1 \le k \le n \le 3000$
- $1 \le x_i \le 10^5$
