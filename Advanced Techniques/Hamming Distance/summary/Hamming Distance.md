# Hamming Distance

**CSES ID:** 2136  
**Full statement:** https://cses.fi/problemset/task/2136  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

The Hamming distance between two equal-length bit strings is the number of positions where they differ. Given $n$ bit strings of length $k$, find the minimum Hamming distance over all pairs of strings.

## Input
First line: $n$ and $k$. Then $n$ lines, each a bit string of length $k$.

## Output
Print the minimum Hamming distance between any two strings.

## Constraints
- $2 \le n \le 2 \cdot 10^4$
- $1 \le k \le 30$
