# Meet in the Middle

**CSES ID:** 1628  
**Full statement:** https://cses.fi/problemset/task/1628  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given an array of $n$ numbers and a target sum $x$, count how many subsets of the array sum to exactly $x$.

## Input
First line: $n$ and $x$. Second line: $n$ integers $t_1,\ldots,t_n$.

## Output
Print the number of subsets summing to $x$.

## Constraints
- $1 \le n \le 40$
- $1 \le x \le 10^9$
- $1 \le t_i \le 10^9$
