# Substring Reversals

**CSES ID:** 2073  
**Full statement:** https://cses.fi/problemset/task/2073  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given a string of length $n$, process $m$ operations that each reverse the substring between positions $a$ and $b$ (1-indexed). Print the final string after all reversals.

## Input
First line: $n$ and $m$. Second line: a string of $n$ uppercase letters. Then $m$ lines, each with two integers $a$ and $b$.

## Output
Print the final string after all operations.

## Constraints
- $1 \le n,m \le 2 \cdot 10^5$
- $1 \le a \le b \le n$
