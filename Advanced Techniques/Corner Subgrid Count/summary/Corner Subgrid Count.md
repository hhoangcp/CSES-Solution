# Corner Subgrid Count

**CSES ID:** 2137  
**Full statement:** https://cses.fi/problemset/task/2137  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given an $n \times n$ binary grid, a subgrid is beautiful when both its height and width are at least two and all four corner cells are black. Count the number of beautiful subgrids.

## Input
First line: $n$. Then $n$ lines of $n$ characters, where `1` is black and `0` is white.

## Output
Print the number of beautiful subgrids.

## Constraints
- $1 \le n \le 3000$
