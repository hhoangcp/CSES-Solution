# Tree Distances II

**CSES ID:** 1133  
**Full statement:** https://cses.fi/problemset/task/1133  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

For every node of a tree you must output the sum of distances from that node to all other nodes.

## Input
First line gives $n$, then $n-1$ edges.

## Output
Print $n$ integers: the sum of distances from each node $1 \dots n$ to all others.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $1 \le a,b \le n$
