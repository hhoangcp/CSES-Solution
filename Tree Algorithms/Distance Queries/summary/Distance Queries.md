# Distance Queries

**CSES ID:** 1135  
**Full statement:** https://cses.fi/problemset/task/1135  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given a tree, answer many queries asking for the graph distance (number of edges) between two nodes.

## Input
First line gives $n$ and $q$. Then $n-1$ edges, followed by $q$ lines each naming two nodes $a$ and $b$.

## Output
Print one integer per query: the distance between the two nodes.

## Constraints
- $1 \le n, q \le 2 \cdot 10^5$
- $1 \le a,b \le n$
