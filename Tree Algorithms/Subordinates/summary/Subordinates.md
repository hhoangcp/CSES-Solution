# Subordinates

**CSES ID:** 1674  
**Full statement:** https://cses.fi/problemset/task/1674  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

A company's hierarchy is given as a tree rooted at the general director (employee 1). For every employee you must output the total number of subordinates beneath them (excluding themselves).

## Input
First line gives $n$. The next line lists, for each employee $2 \dots n$, their direct boss.

## Output
Print $n$ integers: the subordinate count for each employee $1 \dots n$.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
