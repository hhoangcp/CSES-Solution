# Tree Distances I

**CSES ID:** 1132  
**Full statement:** https://cses.fi/problemset/task/1132  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

For each node of a tree you must output the greatest distance from it to any other node.

## Input
First line gives $n$, then $n-1$ edges.

## Output
Print $n$ integers: the maximum distance from each node $1 \dots n$ to another node.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $1 \le a,b \le n$
