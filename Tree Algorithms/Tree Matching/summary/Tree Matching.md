# Tree Matching

**CSES ID:** 1130  
**Full statement:** https://cses.fi/problemset/task/1130  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

A matching in a tree is a set of edges sharing no endpoints. Find the maximum number of edges in such a matching.

## Input
First line gives $n$, then $n-1$ edges.

## Output
Print one integer: the maximum number of edges in a matching.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $1 \le a,b \le n$
