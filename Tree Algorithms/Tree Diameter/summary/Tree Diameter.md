# Tree Diameter

**CSES ID:** 1131  
**Full statement:** https://cses.fi/problemset/task/1131  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

The diameter of a tree is the length (in edges) of the longest simple path between any two nodes. Find it.

## Input
First line gives $n$, then $n-1$ edges.

## Output
Print one integer: the tree's diameter.

## Constraints
- $1 \le n \le 2 \cdot 10^5$
- $1 \le a,b \le n$
