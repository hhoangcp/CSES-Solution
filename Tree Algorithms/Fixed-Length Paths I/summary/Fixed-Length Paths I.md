# Fixed-Length Paths I

**CSES ID:** 2080  
**Full statement:** https://cses.fi/problemset/task/2080  
**Time limit:** 1.00s | **Memory limit:** 512MB

> Problem summary.

Given a tree, count the number of distinct simple paths whose length is exactly $k$ edges.

## Input
First line gives $n$ and $k$. Then $n-1$ edges.

## Output
Print one integer: the number of paths of length exactly $k$.

## Constraints
- $1 \le k \le n \le 2 \cdot 10^5$
- $1 \le a,b \le n$
